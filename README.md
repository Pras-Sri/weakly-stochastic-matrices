# Weakly Stochastic Matrices

This repository contains a research note on weakly stochastic matrices and their applications to graph theory.

## Authors

* Prashanth Sridharan
* Thomas Bier

## Abstract

A matrix is called weakly column stochastic if the all-one vector lies in its column space. Similarly, a matrix is called weakly row stochastic if the all-one vector lies in its row space.

This note develops a criterion for determining weak stochasticity using row difference matrices and applies the resulting theory to adjacency matrices and distance matrices of graphs, particularly trees.

## Terminology Note

The term "weakly stochastic" has been used elsewhere in the literature with a different meaning, typically referring to matrices whose rows or columns sum to one.

The terminology used in this manuscript reflects the original usage adopted during this research project and should not be confused with the standard definition of weakly stochastic matrices found in later literature.
